# ALS
ESP8266 Rgb lamp server

## HARDWARE
### lamp format
- tower 16x14

### strip
- ws2812b

### pic
- wemos d1 mini

## CODE
  - Compil with:
    - Arduino ide
  - Text editor:
    - Sublime-text

- dowload to "\Users\You\Documents\Arduino\libraries"
